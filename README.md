# The Newcastle Night Spy

## Run locally

```bash
cd /Users/craigsmith/newcastle-night-agent
python3 -m http.server 4180
```

Open `http://127.0.0.1:4180`.

## Add Chapter 1 images

Place your two images in:

- `images/chapter-1-image-1.jpg`
- `images/chapter-1-image-2.jpg`
- `images/chapter-1-hero.jpg`

The page is already wired to these filenames.

## Add Chapter 2 images

When ready, place Chapter 2 images in:

- `images/chapter-2-image-1.jpg`
- `images/chapter-2-image-2.jpg`
- `images/chapter-2-image-3.jpg`

`chapter-2-image-1.jpg` is the top image and `chapter-2-image-2.jpg` / `chapter-2-image-3.jpg` are mid-story images.
